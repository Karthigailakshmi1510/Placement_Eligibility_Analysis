Project Description

The Campus Placement Management System is a web application designed to simplify and automate the college placement process. It helps students check eligibility, apply for drives, and allows admins to manage students and placement drives efficiently.

🚀 Features
👨‍🎓 Student
Login using credentials provided by admin
Update profile and upload resume
View available placement drives
Check eligibility for drives
Register for eligible drives
👩‍💼 Admin
Add and manage student details
Post placement drives
Set eligibility criteria (CGPA, department, skills)
View and manage applications
Download eligible students list as PDF
🛠️ Tech Stack
Frontend: React.js, HTML, CSS
Backend: Node.js, Express.js
Database: MongoDB
🔄 Project Flow
Student/Admin logs into the system
Student updates profile and views placement drives
Admin posts drives and sets eligibility criteria
Backend checks student eligibility
Eligible students can register for drives
Admin can download eligible students list in PDF format
🌐 Architecture

User → React Frontend → API → Node.js Backend → MongoDB Database → Response → UI

📂 Modules
Authentication Module
Student Profile Management
Placement Drive Management
Eligibility Checking System
Application & Registration Module
PDF Generation Module
